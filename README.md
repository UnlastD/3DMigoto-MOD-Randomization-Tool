# 3DMigoto-MOD-Randomization-Tool
3DMigoto MOD Randomization Tool，当老玩家3DMigoto MOD文件夹内角色mod过多，苦恼于需要对每个角色手动切换mod时，可尝试此脚本，该工具实现

1.角色 MOD 自动随机启用 / 禁用（默认每个角色mod文件夹仅启用一个mod）

2.角色 MOD子分组管理（对每个角色mod文件夹可再进行子分组，例如发型分组，每次随机仅启用一个，服装分组也会随机仅启用一个，由此可让每个角色mod多启用几组mod）

3.针对 MOD 进行自定义锁定，排除随机化

4.配置预设保存等功能，可快速保存随机化设定预设

5.分为图形界面版（UI） 和 无命令行版（纯脚本） 两个版本，无UI版本启动后按默认预设进行自动处理。

建议先使用UI版本进行自定义随机化管理，保存预设后，后续再用无UI版本处理（无UI版本处理更快更方便，当然直接用UI界面处理也可以，有自动随机化选项）
